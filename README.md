A backend console application built with Java to help users track their gym workouts, calculate total lifted volume, and persist workout history locally.

## Features
* **Dynamic User Input:** Captures exercise names, weights, and repetitions using Java `Scanner`.
* **Volume Calculation:** Automatically calculates the total volume (weight * reps) for each session.
* **Data Persistence:** Uses Java File I/O (`FileWriter`, `PrintWriter`) to save and append workout summaries to a local `.txt` file.
* **Object-Oriented Design:** Structured using OOP principles with `Workout` and `WorkoutSet` models.

## Technologies Used
* Java (JDK)
* File I/O
* Git & GitHub

## How to Run
1. Clone the repository to your local machine.
2. Compile and run the `Main.java` file in the `src/pack1` directory.
3. Follow the console prompts to enter your workout details.
4. Check the `antrenman_gecmisi.txt` file for your saved records.
